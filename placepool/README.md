# PlacePool — University Placement Prep Portal

A full-stack placement preparation portal with:
- ✅ Student signup/login with OTP email verification
- ✅ DSA problem sheet with topic filter, difficulty filter, progress tracking
- ✅ Core subjects (OS, DBMS, CN, OOP) with MCQ/short answer practice
- ✅ Aptitude practice with interactive MCQ and answer reveal
- ✅ Company resources — admin adds resources per company, any file format
- ✅ Progress dashboard with charts, heatmap, streak tracking
- ✅ Admin panel — add/manage all content without touching code
- ✅ File uploads (PDF, Video, PPT, Excel, Images, ZIP) via Cloudinary

## Quick Start
See **SETUP.md** for complete setup instructions.

```bash
# Backend
cd backend && cp .env.example .env  # fill in values
npm install && npm run dev

# Frontend  
cd frontend && cp .env.example .env
npm install && npm start
```
